Términos y Condiciones de Uso del Bot de Discord

El uso del bot RillerCraps en un servidor de Discord requiere la recopilación de ciertos datos de usuario. Los Datos recopilados incluyen, pero no se limitan a, los valores de ID de usuario de Discord. El uso del Bot se considera un acuerdo con los términos de este documento.

Acceso a los Datos

El acceso a los Datos solo está permitido para los desarrolladores del Bot y solo en la medida necesaria para el desarrollo, prueba e implementación de las características del Bot. Los Datos no se venden, proporcionan ni comparten con ningún tercero, excepto cuando lo exija la ley o un acuerdo de Términos de Servicio. Puedes solicitar ver los datos en cualquier momento a través de RillerCraps.

Almacenamiento de Datos

Los Datos se almacenan en una base de datos MongoDB. La base de datos está protegida para evitar el acceso externo, sin embargo, no se proporciona garantía y los propietarios del Bot no asumen responsabilidad por la violación no intencionada o maliciosa de los Datos. En caso de un acceso no autorizado a los Datos, los usuarios serán notificados a través de la aplicación cliente de Discord.

Derechos de los Usuarios

En cualquier momento, tienes el derecho de solicitar ver los Datos relacionados con tu cuenta de Discord. Puedes enviar una solicitud a través del servidor de Discord. También tienes el derecho de solicitar la eliminación de Datos relevantes.

Preguntas

Si tienes alguna pregunta o preocupación acerca de qué datos podrían estar siendo almacenados de tu cuenta, comunícate con nombre_de_tu_bot. Para obtener más información, consulta los Términos de Servicio de Discord.
